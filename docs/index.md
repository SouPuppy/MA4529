ABZU is a layered quantitative systems workspace. `Loom` provides the streaming and event foundation, `Aurum` runs execution and strategy workloads, and `Lyra` manages runs, sessions, and daemonized control paths. The documentation below is organized around those system boundaries so you can move from infrastructure to execution to operations cleanly.
